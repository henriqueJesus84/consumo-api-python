# Consumo de API da NASA com Python

Projeto desenvolvido em Python para consumir dados da API da NASA.

## Sobre
O objetivo do projeto é praticar requisições em APIs, tratamento de dados e exibição das informações.

## Tecnologias
- Python
- Requests

## Funcionalidades
- Consumo de dados da API da NASA
- Tratamento das informações recebidas
- Exibição dos dados

## Como executar
1. Instalar o Python
2. Instalar a biblioteca requests:
pip install requests

3. Executar o arquivo:
python nome_do_arquivo.py

## Objetivo
Praticar integração com APIs e manipulação de dados utilizando Python.
